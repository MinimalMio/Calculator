## Getting Started

Open Calculator type the following thing and press ENTER:

```
let a = 50
50 + 50
```

You can type `help` for more information.

## How to build

You need CMake 3.10+ ,Ninja and G++ Compiler for build this project.

```Bash
mkdir build && cd build
cmake .. && ninja
```
